# Agent 落地图鉴

不讲概念，只看真实任务——AI Agent 究竟在帮哪些人做哪些事。

**在线访问：** https://jerryhub366.github.io/study-agent/

## 这是什么

一个收录 AI Agent 真实落地案例的交互式图鉴。覆盖软件开发、客户服务、法律、科研、内容媒体、金融、个人效率、运营电商、教育等领域，每个案例包含：

- 具体任务场景（以前 vs 现在）
- 使用的工具/产品
- 入门门槛标注
- 来源引用链接

## 特性

- 按行业分类筛选
- 纯静态，无构建步骤，GitHub Pages 直接部署
- 移动端适配

## 本地运行

直接用浏览器打开 `index.html`，或启动任意静态服务器：

```bash
npx serve .
```

## 技术栈

- 纯 HTML / CSS / JavaScript
- 零依赖，单页应用
